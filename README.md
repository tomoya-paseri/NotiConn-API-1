# How to start
```
git clone https://github.com/KatsuyaAkasaka/NotiConn-API

cd path/to/NotiConn-API

direnv edit .

hearing me to env variables

yarn install
```

# commands
```
make offline 
- start offline

make deploy
- deploy

make remove
- remove lambda from aws

make save-local
- run save function local
```
